# 🌦 Advanced Weather Application  
## OIBSIP - Oasis Infobyte Internship

This project is developed as part of the Oasis Infobyte Internship Program.

---

## 📌 Project Overview

The Advanced Weather Application is a GUI-based desktop application built using Python and Tkinter.  
It fetches real-time weather data and 5-day forecast using the OpenWeatherMap API.

The application also supports:
- Current location weather detection (via IP)
- Temperature unit toggle (°C / °F)
- Weather icons
- 5-day forecast display

---

## 🛠 Technologies Used

- Python
- Tkinter (GUI)
- Requests (API calls)
- Pillow (Image handling)
- Geocoder (Location detection)
- OpenWeatherMap API

---

## ✨ Features

- 🌍 Search weather by city name  
- 📍 Get weather using current location  
- 🌡 Toggle between Celsius and Fahrenheit  
- 📅 5-Day weather forecast  
- 🖼 Weather condition icons  
- ⏱ Real-time update timestamp  

---

## ⚙ Installation & Setup

1. Clone this repository:

git clone <your-repo-link>

2. Install required libraries:

pip install -r requirements.txt

3. Add your OpenWeatherMap API key inside:

API_KEY = "YOUR_API_KEY"

4. Run the application:

python main.py
